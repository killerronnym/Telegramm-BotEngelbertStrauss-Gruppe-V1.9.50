Wichtige Gruppen-Vorbereitung:
  • 1. Haupt-Gruppe: Erstelle eine Gruppe für deine Community und füge den Bot als Administrator hinzu.
  • 2. Admin-Gruppe: Erstelle eine separate Gruppe nur für Moderatoren/Admins und lade den Bot ebenfalls ein. Hier landen wichtige interne System-Meldungen.
So findest du die IDs:
  1. Sende /id in die jeweilige Gruppe.
  2. Klicke unten auf , um die ID automatisch zu übernehmen.

Bot Token erstellen
BotFather finden

Suche in Telegram nach dem offiziellen @BotFather Account und klicke unten auf den Start Button, um den Bot zu aktivieren.

Web-App öffnen

Klicke unten links im Chat auf das lila Viereck-Icon (Menü), um das moderne Web-Interface von BotFather zu öffnen.

Bot erstellen

Wähle in der Übersicht den großen Button "+ Create a New Bot" aus, um den Erstellungsprozess zu starten.

Details ausfüllen

Gib deinem Bot einen Anzeigenamen und einen eindeutigen Nutzernamen (dieser muss auf _bot enden). Wenn du fertig bist, klicke unten auf den blauen CREATE BOT Button.

Token speichern

Dein Bot wurde erfolgreich erstellt! Kopiere dir den langen HTTP API Token (siehe Bild) und füge ihn hier im Installations-Setup ein. Bewahre diesen Token gut auf!

Sicherheitshinweis: Dein Token ist wie ein Passwort! Gib ihn niemals an Fremde weiter, da man damit volle Kontrolle über deinen Bot hat.
